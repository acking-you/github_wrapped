# Cookbook (Optional Recipes)

This folder contains optional, deeper implementation recipes.
It is **not** mandatory reading unless you are implementing that specific feature.

## Recipes

- `cookbook/scroll_input.md` — wheel paging vs inner scroll, Spacebar rules, focus hygiene
- `cookbook/draggable_hud.md` — draggable floating controls without breaking clicks
- `cookbook/soundcloud_player.md` — SoundCloud widget integration (autoplay fallback, seek offset, focus issues)
